### Setup

To run on localhost:

```bash
npm run dev
```

To build:

```bash
npm run build
```

To create a component (Atom, Molecule, Organism):

```bash
npm run plop
```

---

To fix vulnerabilities:

```bash
npm audit fix
```

To install dependencies:

```bash
npm install
```
